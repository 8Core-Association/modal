SEUP Modul

Sustav elektroničkog uredskog poslovanja omogućava potpunu digitalnu evidenciju dokumenata.

Brzi početak

  1. Aktivirajte SEUP modul u Dolibarr administraciji
  2. Konfigurirajte oznake ustanove u postavkama
  3. Definirajte klasifikacijske oznake
  4. Kreirajte prvi predmet

Rad s predmetima

Predmeti su osnovne jedinice evidencije u SEUP sustavu. Svaki predmet može sadržavati:

  • Osnovne podatke (naslov, opis, klasifikacija)
  • Akte i priloge
  • Zaprimanja i otpreme
  • Suradnike i tagove

Instalacija

Sistemski zahtjevi

  • PHP 7.4 ili noviji
  • MySQL 5.7 ili MariaDB 10.3+
  • Dolibarr 15.0 ili noviji
  • Apache/Nginx web server

Postupak instalacije

1. Raspakujte modul u htdocs/custom/seup
2. Prijavite se kao administrator
3. Idite na Home → Setup → Modules
4. Pronađite SEUP i kliknite "Activate"
5. Konfigurirajte postavke u Setup → Modules → SEUP

Konfiguracija

Osnovne postavke

U postavkama modula možete konfigurirati:

  • Oznaku ustanove (OMAT)
  • Internu oznaku korisnika
  • Nextcloud integraciju
  • Šablone dokumenata

Integracije

Nextcloud

SEUP se može integrirati s Nextcloud cloud sustavom za skladištenje dokumenata.

Upute za konfiguraciju:

  1. Unesite Nextcloud URL
  2. Generirajte App Password
  3. Testirajte konekciju
